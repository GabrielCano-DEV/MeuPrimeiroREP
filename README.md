# Guia do GitHub
## 1°PASSO
### Criar um repositório no GitHub e copiar o URL
## 2°PASSO
### Escolher ou criar uma pasta para guardar o repositório
## 3°PASSO
### Abrir a pasta no terminal para executar o código:
## "git clone (URL que foi copiada no primeiro passo)"
## 4°PASSO
### Usar o comando: 
## "dir" para verificar se o diretório foi adicionado
## 5°PASSO
### Entre no diretório digitando o comando "cd (Nome da pasta)"; após isso digite o comando "code ." para abrir no VScode
## 6°PASSO
### Digitar o comando: 
## "git add ."
### Para adicionar todos os arquivos.
## 7°PASSO
### Digitar o comando:
## "git commit -m "(escrever a alterção feita ex: "Adicionei meu nome")"
## 8°PASSO
### Digitar o comando:
## "git push -u origin main" para enviar as alterações para o GitHub.
## 9°PASSO
### Usar codigo:
## "Git pull"
### Para atualizar as mudanças à maquina local, pois quando o arquivo é mudado no GitHub, a máquina local fica desatualizada.
